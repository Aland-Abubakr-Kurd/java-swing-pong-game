# • Java Swing Pong Game

A classic Pong game implemented in Java Swing.  
The game features two paddles, a bouncing ball, a scoring system, and keyboard controls.

---

## • Features

- Two-player controls:
  - Player 1: W (up), S (down)
  - Player 2: Up Arrow (up), Down Arrow (down)
- Ball speed increases after hitting a paddle
- Score tracking with a visual net in the middle
- Double buffering to prevent screen flicker
- Continuous game loop at ~60 FPS
- Simple, colorful graphics for paddles, ball, and score

---

## Technologies Used

- Java
- Java Swing (GUI)
- AWT (Events, Graphics)
- Threads (Runnable for game loop)
